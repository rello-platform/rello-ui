# Color System

All colors are defined as CSS custom properties in `design-tokens.css`. Brand colors are customizable per tenant; all other colors remain constant for UI consistency.

These change per tenant via CSS variable overrides.

## Semantic Aliases

| Token | Maps To | Use |
|-------|---------|-----|
| `--background` | `--neutral-50` | Page background |
| `--foreground` | `--neutral-900` | Primary text |
| `--card-background` | `#FFFFFF` | Card surfaces |
| `--card-border` | `--neutral-100` | Card borders |
| `--muted` | `--neutral-100` | Muted backgrounds |
| `--muted-foreground` | `--neutral-500` | Secondary text |
