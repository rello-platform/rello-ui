## CSS Classes

| Class | Font Family | Use |
|-------|-------------|-----|
| `.font-heading` | Montserrat | Section headers, titles |
| `.font-body` | Open Sans | Body text, descriptions |
| `.font-ui` | Hind | Buttons, badges, labels |
| `.font-stat` | Mina | Stats, scores, metrics |
