## Transitions

| Token | Duration | Use |
|-------|----------|-----|
| `--transition-fast` | 150ms | Micro-interactions (hover, toggle) |
| `--transition-normal` | 200ms | Standard transitions |
| `--transition-slow` | 300ms | Complex animations (modals, drawers) |

## Z-Index Scale

| Token | Value | Use |
|-------|-------|-----|
| `--z-dropdown` | 10 | Dropdown menus |
| `--z-sticky` | 20 | Sticky headers |
| `--z-fixed` | 30 | Fixed elements |
| `--z-overlay` | 40 | Overlays |
| `--z-modal` | 50 | Modals, dialogs |
| `--z-popover` | 60 | Popovers |
| `--z-tooltip` | 70 | Tooltips |
